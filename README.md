Estructura del Proyecto

El código está diseñado para simular una experiencia básica de compra en línea. Se estructura de la siguiente manera:

Verificación de edad: Se solicita la edad del usuario y se usa un bloque if para determinar si puede continuar con la compra.

Selección de productos: Si cumple con la edad mínima, se le permite ingresar hasta dos productos con sus precios.

Guardado de artículos: Se utiliza un bucle for con un límite de 2 iteraciones para permitir la selección de productos sin hacerlo demasiado extenso.

Cálculo del total: Se suman los precios de los productos seleccionados y se muestra el total de la compra.

Restricción por edad: Si el usuario es menor de edad, se muestra un mensaje indicando que no puede realizar la compra.

Herramientas Utilizadas

JavaScript básico.

Estructuras de control (if-else).

Bucle for para gestionar la selección de productos.

Manipulación de entradas del usuario con prompt().

Conversión de tipos con parseInt() para manejar precios.
